# FLOWER-CLASSIFICATION
Flower classification using MobileNetV3 and EfficientNetV2 deep learning models

## Overview
This project implements an advanced flower classification system using state-of-the-art deep learning models. The system accurately identifies and classifies flowers into 5 distinct categories (tulip, rose, daisy, dandelion, sunflower) using transfer learning with MobileNetV3 and EfficientNetV2 architectures. The project demonstrates the practical application of convolutional neural networks for computer vision tasks in botanical classification.
